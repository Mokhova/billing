Статистика по конкретному объекту

Конкретный счет, конкретная связь, конкретный клиент и т.д.
Это нужно, чтобы понять, насколько плотно взаимодействуют с конкретным объектом, как часто. Таким методом пользуемся только тогда, когда в урле нет id объекта.
Поэтому привисываем в action какой-то идентификатор объекта.

отдельно с id Category: client-connection
Action: <название действия>-id-<идентификатор>
Label: create-link

Общая статистика по сценарию

Все события отправляем по какому-то завершающему действию (сабмит, показать, т.д.), чтобы не засорять аналитику тем, что пользователь перетыкивает по 10 раз. Нас интересует конечный набор параметров.

в кучу Category: client-connection
Action: <название действия>
Label: create-link
