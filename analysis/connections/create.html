Создать связь из блока контакты

Сколько людей создают новые связи, несмотря на миграцию КС.

Событие в GA
спека _trackEvent(category, action, opt_label, opt_value, opt_noninteraction) Category: client-connection
Action: create-open
Label: create-link
отправить сразу

Отмена создания связи

Посмотрим, может кто-то не понял, что это за панелька, что в ней делать =)

Событие в GA
спека _trackEvent(category, action, opt_label, opt_value, opt_noninteraction) Category: client-connection
Action: create-cancel
Label: cancel-link
отправить сразу

Сколько связей создано?

Когда появляется статус «связь создана» отправляем два события.

Событие в GA
спека Category: client-connection
Action: create-created
Label: created
отдельно с id в кучу

Данные созданной связи

Отправляем по кнопке «показать».

Событие в GA
спека Category: client-connection
Action: create

Label: clients-N //Кол-во клиентов в связи
Label: star //Выбран «звездный» клиент
Label: contact //Выбрано КЛ
Label: contact-empty //Не выбрано КЛ
Label: product-<название продукта> //Каждый продукт отдельно
Label: comment //Заполнен коммент
отдельно с id в кучу

Процесс создания связи

Посмотрим, удаляют ли:

  • Добавленные организации
  • Выбранное контактное лицо

Событие в GA
спека Category: client-connection
Action: create
Label: delete-client //ссылка «удалить» у клиента
Label: delete-contact //ссылка «удалить» у контакта
отдельно с id в кучу отправить сразу

Ошибки

Сколько возникает ошибка и что дальше при этом делают.

Событие в GA
спека Category: client-connection
Action: create
Label: delete-message //считаем, что показалось сообщение
Label: delete-and-close //нажали на «удалить и закрыть»
отдельно с id в кучу отправить сразу
