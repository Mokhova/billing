Метрики отчетов

Отчет по счетам, оплатам и сделкам

Хотим понимать, какие отчеты популярные, и какие параметры чаще выбирают. Возможно будем запоминать необходимые. По каждому отчету надо видеть отдельно:

  • С какими именно датами грузили. Например, 1.08.2016
  • Какой диапазон выбирают: 10 дней, 15 дней.

Событие в GA
спека Category: report-bills // отчет по выставленным счетам
Category: report-payment // отчет по оплаченным счетам
Category: report-deals // отчет по завершеннм сделкм
Action: download
Label:
— N // N—это число дней, посчитать, сколько дней выбрали в попапе периода.
— dd.mm.yyyy—dd.mm.yyyy или mm.yyyy // что конкретно выбрали в попапе
— product-<имя продукта> // каждый продукт, который выбирали
Category: report-pp // отчет по потенциальным продажам
Action: download
Label:
— N // N—это число дней, посчитать, сколько дней выбрали в попапе периода.
— dd.mm.yyyy—dd.mm.yyyy или mm.yyyy // что конкретно выбрали в попапе
— product-<имя продукта> // каждый продукт, который выбирали
— source-<имя источника> // источник, который выбирали в выпадашке
