Привязка и выставление счета

Хочется понять, сколько счетов привязывают, выставляют ли счет из ПП.
Что происходит чаще.

Событие в GA
спека Category: pp-cycle-actions

Action: create-bill // нажали на панельке «выставить счет»
Action: bind-bill // нажали на панельке «привязать счет»

Label: button
отдельно с id в кучу

Результат выставления счета

Хочется видеть, выставили счет или нет.

Событие в GA
спека Category: pp-cycle-actions
Action: create-bill
Label: created // когда нажали в мастере «выставить счет на...»
Label: close // когда закрыли мастер
отдельно с id в кучу

Результат привязки счетов

Если привязывают счета, то сколько за раз.

Событие в GA
спека Category: pp-cycle-actions
Action: bind-bill
Label: bound-N // нажали в лайтбоксе «привязать», N—кол-во привязанных счетов
Label: cancel
Label: close
отдельно с id в кучу

Отказ

Хочется понимать, как часто отказывают и с какой причиной, пишут ли комментарий

Событие в GA
спека Category: pp-cycle-actions
Action: finish-refused
Label: reason-<причина>
Label: comment-true // если написали комментарий
Label: comment-empty // без комментариев
Label: cancel
Label: close
отдельно с id в кучу
