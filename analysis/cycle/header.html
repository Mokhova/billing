Дата в шапке

Посмотрим, нажимают ли календарик в шапке.
Хотим узнать, на какой срок откладывают обычно: неделя, месяц, пара часов. Для этого будем посылать разницу между текущей датой и датой отложенности в днях. Если даты совпадают и указали время, то отправляем, на сколько часов отложили.

Событие в GA
спека Category: pp-cycle-header
Action: postpone
Label: calendar // кликнули на дату с календарем
Label: submit // нажали «отложить», время не указали
Label: submit-<срок отложенности> // нажали «отложить», отправляем срок
Label: cancel // нажали «отменить»
Label: close // нажали на кресик
отправить сразу отдельно с id в кучу
